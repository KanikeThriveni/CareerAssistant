# Career Assistant Agent

This project is a web application that allows end users to upload their resumes (PDF/DOCX), parses the resume to extract relevant information (skills, experience, education), and provides career guidance or recommendations based on the extracted data.

## Features
- Resume upload (PDF/DOCX)
- Resume parsing
- Career guidance and recommendations

## Tech Stack
- Python
- Flask
- Frontend: HTML/CSS/JS (basic)

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`

## Folder Structure
- `app.py`: Main Flask application
- `templates/`: HTML templates
- `static/`: Static files (CSS/JS)
- `resume_parser.py`: Resume parsing logic
- `requirements.txt`: Python dependencies
